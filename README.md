# Few typescript examples

### using node 17 installed with nvm
`nvm use 17`

### install typescript globally if you want or don't
`npm i -g typescript`
`npm i typescript`

### install other packages
`npm i`

### compile the ts
`tsc test.ts`

### run the js file with node
`node test.js`
